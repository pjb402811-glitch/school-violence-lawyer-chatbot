# ⚖️ 쏭비서 학폭 전문 AI 변호사 챗봇 V2.0 (Vercel Serverless 배포판)

본 프로젝트는 학교폭력 처분 결과 이후에도 한 교실에 가해자와 방치된 자녀를 긴급 구제하고, 진행 중인 수사 단계의 승기를 쟁취할 수 있도록 돕는 **초고품격 RAG(검색 증강 생성) 기반 AI 법률 상담 웹앱**입니다. 

Vercel의 Serverless Functions(Python FastAPI)와 현대적 Glassmorphism UI를 결합하여, **서버 비용 0원으로 24시간 가동되며 휴대폰과 PC에서 앱처럼 자유롭게 사용**할 수 있습니다.

## 📱 주요 탑재 기능 요약

1. **2차 고도화 수록 법리/판례 학습**:
   * **대법원 2022두56676 판결**: "가해학생 학습권보다 피해자의 보호구제가 압도적 최우선" 법리 완벽 탑재.
   * **정신적 상해 상당인과관계**: 신체 상해가 없더라도 정신과 치료(PTSD) 및 등교 거부 상태가 중대한 폭력 결과임을 인정하는 중앙행심위 재결례 탑재.
   * **학급교체 기각 성공 사례**: 가해자가 낸 반 분리 처분 취소 청구를 기각시킨 실재 성공 판례 수록.
   * **NLP 감정 분류 케어**: 깃허브 `osondoson-nlp` 또래 상담의 NLP 흐름을 흉내 내어 질문 속 감정적 고통(우울, 등교 거부)을 다중 분석 및 가중 피드백하는 로직 내장.
2. **교육부 고시 기준 학폭위 예상 징계 시뮬레이터**:
   * 심각성, 지속성, 고의성, 반성도, 화해정도(각 0~4점)를 슬라이더로 조절하면 실시간으로 예상 조치(1~9호)를 자동 연산합니다.
3. **실전 서식 4종 원클릭 즉시 다운로드**:
   * 학교장 분리 촉구서, 행정심판 청구서, 수사기관의견서, 증거수집 가이드를 마크다운(.md) 파일로 즉시 다운로드하여 바로 인쇄 및 이송할 수 있습니다.
4. **쏭비서 에이전트 전용 앱 아이콘**:
   * 황금빛 천칭과 네온 방패가 어우러진 현대적이고 세련된 전용 앱 로고(`public/app_icon.png`)가 탑재되어 모바일 바탕화면 추가 시 품격을 완성합니다.

## 🚀 Vercel 5분 만에 무료 배포하는 방법 (휴대폰/컴퓨터 공용)

Vercel에 배포하면 평생 무료로 나만의 주소(`https://project-name.vercel.app`)를 획득하여 모바일과 PC에서 자유롭게 사용할 수 있습니다.

### 1단계: GitHub 리포지토리 생성 및 업로드
1. 개인 **GitHub** 계정에 접속하여 새로운 Public 또는 Private 리포지토리를 생성합니다.
2. 본 프로젝트 폴더(`변호사 챗봇`)의 모든 파일을 Git을 통해 깃허브에 push합니다:
   ```bash
   git init
   git add .
   git commit -m "feat: deploy school violence legal chatbot to vercel"
   git branch -M main
   git remote add origin <대표님의_깃허브_레포지토리_주소>
   git push -u origin main
   ```

### 2단계: Vercel에서 1초 만에 Import
1. [Vercel 공식 홈페이지](https://vercel.com)에 가입/로그인합니다. (GitHub 계정 연동 권장)
2. 대시보드 우측 상단의 **[Add New] -> [Project]** 단추를 누릅니다.
3. 1단계에서 업로드한 깃허브 리포지토리를 목록에서 찾아 **[Import]**를 클릭합니다.
4. 별도의 설정 변경 없이 **[Deploy]** 단추를 누릅니다.
   * Vercel의 빌드 엔진이 `vercel.json`을 읽고 static 파일(HTML/CSS/JS)과 Python serverless API를 알아서 자동 빌드 및 배포합니다.
5. 배포 완료 시 제공되는 `https://...vercel.app` 고유 도메인 주소로 즉시 전 세계 어디서나 접속이 가능합니다!

## 📲 스마트폰 홈 화면에 '전용 앱(PWA)'으로 등록하기

배포된 웹페이지를 스마트폰 홈 화면에 아이콘으로 얹어 마치 네이티브 앱처럼 사용할 수 있습니다.

### 🍏 iPhone (iOS / Safari)
1. **Safari** 브라우저를 열고 대표님의 Vercel 배포 주소로 접속합니다.
2. 브라우저 하단의 **[공유]** 버튼(네모 안의 화살표)을 누릅니다.
3. 메뉴를 아래로 스크롤하여 **[홈 화면에 추가]**를 선택합니다.
4. 이름("학폭 AI 변호사")을 지정하고 추가를 누르면, 바탕화면에 **쏭비서 법률 에이전트 전용 황금 천칭 로고**가 새겨진 앱이 즉시 생성됩니다.

### 🤖 Android (갤럭시 / Chrome)
1. **Chrome** 브라우저를 열고 Vercel 배포 주소로 접속합니다.
2. 우측 상단의 **[더보기(점 3개)]** 아이콘을 누릅니다.
3. **[홈 화면에 추가]** 또는 **[앱 설치]**를 클릭합니다.
4. 바탕화면에 전용 아이콘과 함께 단독 앱으로 설치 완료됩니다.
